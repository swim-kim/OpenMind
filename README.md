## 😄 프로젝트 소개
### 📄 프로젝트 설명 
질문과 답변을 통해 마음을 열고 대화 나누는 소통 플랫폼인 '오픈마인드' 서비스
### 🪄 주요 기능
- 개별 피드 생성 : 개별피드 카카오/페이스북/링크 공유를 통해서 다른 유저가 접속해서 질문 작성 가능
- 질문 답변 : 피드를 생성한 유저가 해당 페이지에서 질문에 답변 가능 (답변 거절도 가능)
- 질문 좋아요/싫어요 : 질문에 좋아요/싫어요를 달 수 있음
